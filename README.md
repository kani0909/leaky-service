# Leaky Service - Демонстрация утечки памяти

Spring Boot приложение, демонстрирующее классическую утечку памяти через бесконечный кэш.

## ⚠️ ВНИМАНИЕ
Этот проект создан **только для образовательных целей**! Он содержит намеренную утечку памяти.

## Проблема
- `ConcurrentHashMap` используется как бесконечный кэш
- Каждая транзакция добавляется в кэш и никогда не удаляется
- Каждый объект весит ~50KB (из-за вложенных объектов и byte[])
- При 1000 запросах кэш занимает ~50MB памяти

## Технологии
- Java 17
- Spring Boot
- Maven
- Lombok
